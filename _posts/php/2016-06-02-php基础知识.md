#### 遍历修改无效

```
foreach ($items as $value) {
	$value['reg'] = 10000;
}
```

修改为

```
foreach ($items as &$value) {
	$value['reg'] = 10000;
}
```

方法同理

```
function fun(&$items){}
```

#### 不定参数

```
public function on()
{
    if(func_get_arg(0) === 'workerStart')
    {
       $this->worker->onWorkerStart = func_get_arg(1);
       return;
    }
    return call_user_func_array(array($this->sockets, 'on'), func_get_args());
} 
```

调用


```
$sender_io->on('connection', function($socket){
	...
});
```
