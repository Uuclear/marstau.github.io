##实例

####匹配QQ号


```
[1-9][0-9]{4,}
```

####匹配中文

```
"^[\u4e00-\u9fa5]{0,}$"
```

####匹配双字节字符

```
([^\x00-\xff]*)
```

e语言:

```
[\u4e00-\u9fa5]+
```
