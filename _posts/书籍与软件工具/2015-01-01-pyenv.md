## COMMANDS

#### MAC切换版本
```
pyenv versions
pyenv local 3.8.2 # 当前目录及其目录切换
python -V # 验证一下是否切换成功
export PYENV_ROOT=~/.pyenv
export PATH=$PYENV_ROOT/shims:$PATH
```
